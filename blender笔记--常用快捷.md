## blender笔记-->常用快捷

| 按键           | 作用                           |
| -------------- | ------------------------------ |
| ~              | 打开视图控制面板               |
| shift+鼠标中键 | 平移视图                       |
| shift+a        | 创建节点、物体                 |
| 1、2、3        | 点、线、面模式                 |
| tab            | 切换物体、编辑模式             |
| e              | 挤出                           |
| g              | 移动                           |
| n              | 工具                           |
| ctrl+r         | 细分                           |
| shift+e        | 打开3d游标控制面板             |
| shift+s        | 打开对齐控制面板               |
| ctrl+p         | （编辑模式下）打开创建父级菜单 |
| ctrl+a         | （物体模式下）打开应用变换菜单 |
| i              | （姿态模式下）打开关键帧菜单   |
| ctrl+shift+v   | （姿态模式下）反转姿势         |
| s              | （姿态模式下）缩放时间轴       |
|                |                                |
|                |                                |
|                |                                |
